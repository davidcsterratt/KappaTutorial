Rule based modelling with Kappa tutorial: software installation instructions

Please follow the installation notes below to prepare for the Kappa tutorial. Note that the KaSim Kappa simulator is a command line tool, so some familiarity with using the command shell on your computer would be good:

  1. Download the KaSim Kappa simulator v2.01
    • MacOSX (10.6)
    • Windows
    • Linux
    The manual for the KaSim simulator is available here.
  2. Open up a command shell on your computer and navigate to where KaSim was downloaded.
  3. Check KaSim is running correctly by entering the command:

    ./KaSim_2.01_linux_x86 (or the equivalent for the version you downloaded).

    You should get the following as a response:

    KaSim 2.01-110112:
    Usage is KaSim -i input_file [-e events | -t time] [-p points] [-o output_file]

  4. Download and save the following Kappa model file (to the same place you put KaSim): installTest.ka
  5. Test running the model by entering the command:

    ./KaSim_2.01_linux_x86 -i installTest.ka -e 1000 -p 1000 (or the equivalent for the version you downloaded).

    You should get something similar to the following as a response:

    Parsing installTest.ka...
    done
    +Self seeding...
    +Initialized random number generator with seed 59585578
    + Compiling...
    -agent signatures
    -variable declarations
    -initial conditions
    -rules
    -observables
    -perturbations
    Done
    + Analyzing non local patterns...
    + Building initial simulation state...
    -Counting initial local patterns...
    Done

    ____________________________________________________________
    ############################################################
    Simulation ended (eff.: 0.993049)

    You should also have a new output file data.out.

Only two things left. An editor for the Kappa model files, and a means of viewing the output of the KaSim simulations. For editing the Kappa models, most users use their own preferred text editor (emacs,notepad,etc.). For viewing the Kappa data output files, again users use their own preferred graphing tool (gnuplot, matlab, excel, etc.). For the purposes of this tutorial, there is a simple java based data plotting tool available if you prefer. To install it:

  1. Download the Kappa PlotViewer v1.1.2 executable jar file.
  2. Run the file as an executable jar using java. Double clicking on the download usually works. Depending on your operating system, you may need to set the downloaded file as executable.
  3. You should see a graphing window appear. Installation successful.
